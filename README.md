## 본 프로그램의 특징
본 프로그램은 다양한 포맷의 원본 미디어 파일(MP4, M4A 등)을 입력받아, 1) 음성인식에 최적화된 표준 오디오(WAV)로 변환하고, 2) 변환된 오디오를 AI 모델로 자동 전사하는 2단계 파이프라인 유틸리티입니다.

## 주요기능
### 1단계: 오디오 전처리 (audio_to_wav.py)
- 미디어 파일 일괄 변환 및 표준화: 지정된 디렉토리 내 미디어 파일(MP4, M4A 등)을 재귀 탐색하여 FFmpeg로 표준 WAV(16kHz, 모노, pcm_s16le, 음량 정규화) 포맷으로 변환. 원본 디렉토리 구조 유지.
- 고속 병렬 처리: 시스템의 모든 CPU 코어를 활용한 멀티프로세싱으로 대용량 파일 고속 변환.
- 리포팅 및 오류 관리: 변환 실패 로그(CSV) 저장 및 변환 완료된 전체 오디오의 총 재생 시간 리포트.

### 2단계: ASR 처리 (run_asr_whisper-large-v3.py)
- 대용량 ASR 일괄 처리: .wav 파일을 재귀 탐색하고 오디오 길이순으로 정렬하여 배치를 최적화. 다중 GPU 병렬 처리를 통해 고속 전사 수행. ASR 결과는 원본 구조와 동일하게 .txt 파일로 저장.
- 자동 평가 기능 (선택 사항): 정답(JSON)과 ASR 결과(.txt)를 비교하여 WER/CER을 자동 계산하고 CSV 리포트 생성.
- 작업 관리: 이미 처리된 파일은 건너뛰어 중복 작업을 방지하며, 오류 발생 시 로그 파일에 기록.
